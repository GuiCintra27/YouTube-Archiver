# Documentação local (WIP)

Esta pasta é reservada para notas de trabalho de curto prazo e planejamento em andamento.

## Regras de uso

- Manter aqui apenas materiais de implementação ativa.
- Ao concluir a implementação, mover os arquivos para `docs/local/archive/`.
- Não usar esta pasta como documentação oficial para recrutadores ou onboarding.

## Onde publicar documentação oficial

- Use `docs/project/` para documentação oficial.
- Comece por `docs/project/INDEX.md`.
